## Push notifications with One Signal - Expo app 

Simple expo app one signal set up for push notifications in the managed workflow, using:
- Expo and One Signal
- EAS Services
- one-signal-expo-plugin
- react-native-onesignal
- expo-constants
- expo-dev-client
